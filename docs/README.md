# Asteroid Destroyer

A fast-paced arcade shooter with endless progression and complex boss mechanics. Experience distinct game phases, evolving enemies, and intense boss battles.

## Demo
Play the live demo at: [https://asteroid.teaserverse.online](https://asteroid.teaserverse.online)

## Quick Start

Follow these steps to set up the project locally:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd asteroid-destroyer
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the game:**
    The terminal will provide a local URL (usually `http://localhost:5173`). Open this link in your browser to play.
