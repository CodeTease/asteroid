# Asteroid Destroyer

A fast-paced browser arcade shooter featuring upgrade systems, boss battles, and an endless Void Mode. Built with Vanilla JavaScript and Vite.

## Gameplay

Survive waves of asteroids and enemies. Collect points to upgrade weapons, unlock AI allies, and defeat the Final Boss to enter the Void.

## Controls

* **Move:** Arrow Keys, WASD, or Touch
* **Aim & Shoot:** Mouse Cursor or Touch Drag
* **Skills:** UI Buttons (Void Mode)

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start local development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Stack

* HTML5 Canvas
* Vanilla JavaScript (ES Modules)
* Vite
* CSS3
